Modelo de regresin lineal simple

Para crear un modelo de regresin lineal simple con relacin al nivel de felicidad se utilizar la variable GDP. Se eligi esta variable porque resulta interesante ver si realmente hay alguna relacin entre el producto interno bruto total de un pas y el nivel de felicidad de sus ciudadanos. Ya que de tener una fuerte correlacin positiva (conforme el GDP crece tambin aumenta el nivel de felicidad) se debera considerar establecer como prioridad el crecimiento del GDP de un pas ya que esto tendr como consecuencia el bienestar y felicidad de su poblacin.

En el siguiente bloque de cdigo se muestra cmo se utilizarn estas 2 variables para crear un modelo de regresin mltiple para posteriormente analizarlas y poder llegar a una conclusin sobre su relacin.

Para abordar el problema del gran rango presente en los valores de GDP se utiliza la funcin np.log.

Se utiliza LinearRegression() de la librera sklearn.linear_model, que ayudar a ajustar el modelo de manera ms eficiente, sin tener que hacerlo manual. Posteriormente se definen los valores para x y y, siendo x el GDP y y el nivel de felicidad. Luego se guarda el resultado en model. Despus se usa la funcin fit para ajustar el modelo a los datos a analizar. Esta funcin slo acepta las variables en forma de arreglo, por lo que se cambia el formato de stas usando array junto con reshape, con los parmetros -1 (la funcin determina la cantidad de filas) y 1 (un arreglo de 1 columna). Por ltimo, se usa print para imprimir los coeficientes ptimos obtenidos a travs de esta funcin, usando intercept_ y coef_.

Para tener un representacin grfica de este modelo se importa la librera matplotlib.pyplot. Se utiliza la funcin predict() para crear los valores de prediccin del modelo. Como los datos puede que no estn en orden, se utiliza la funcin argsort para que acomode los valores en el arreglo. Posteriormente se utiliza la funcin scatter para aadir los puntos correspondientes a logGDP y la Felicidad. Luego se utiliza la funcin plot para graficar la lnea de tendencia que se calcul con el modelo, usando los valores de logGDP en orden y los valores que se predijeron de la Felicidad con el modelo, tambin en orden.

Por ltimo, en el bloque de cdigo se aade la librera r2_score para calcular el valor de $R^2$ del modelo y hacer un mejor anlisis.

import numpy as np
from sklearn.linear_model import LinearRegression
import matplotlib.pyplot as plt

x_log = np.log(datos["GDP"])
y = datos["Felicidad"]
model_log = LinearRegression()
model_log.fit(np.array(x_log).reshape(-1,1), y)

#Intercerpto y Coeficient
print("Intercepto:", model_log.intercept_)
print("Coeficiente:", model_log.coef_[0])

#Grfica
y_pred = model_log.predict(np.array(x_log).reshape(-1,1))

indice_orden = np.argsort(x_log)
x_log_sorted = x_log.iloc[indice_orden]
y_pred_sorted = y_pred[indice_orden]

plt.figure(figsize=(7,5))
plt.scatter(x_log, y, alpha=0.5, label="Datos")
plt.plot(x_log_sorted, y_pred_sorted, color="red", label="Regresin lineal")
plt.xlabel("log(GDP)")
plt.ylabel("Nivel de felicidad")
plt.title("Regresin lineal: Felicidad vs log(GDP)")
plt.legend()
plt.show()

#R2
from sklearn.metrics import r2_score

r2 = r2_score(y, y_pred)
print("R:", r2)

Limitaciones

Algunas de las limitaciones de este anlisis fueron la poca cantidad de pases que quedaron una vez que se eliminaron a aquellos que no tenan todos los datos. De igual manera, otro obstculo presente en este reporte fue que no todos los datos eran del mismo ao. Por ejemplo, el GDP corresponde al 2020, ao de la pandemia que pudo haber alterado los resultados finales. Mientras que la felicidad, tasa de fertilidad y esperanza de vida fueron datos obtenidos para el 2022. Al mismo tiempo que la densidad poblacional corresponde a los valores del ao 2021. Esto causa que el modelo final y la relacin entre variables no sea del todo real, ya que las situaciones de cada pas, y del mundo en general, pueden cambiar mucho ao con ao.
