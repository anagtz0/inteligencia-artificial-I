## Regresión Lineal

En este proyecto se trabaja con 2 tipos de regresiones diferentes: regresión lineal simple y regresión lineal múltiple. Se hizo un modelo para ver qué tanto se podía predecir la variazción de los niveles de felicidad considerando solo el GDP. Posteriormente se hizo un nuevo modelo agregando las variables "tasa de fertilidad", "esperanza de vida" y "densidad poblacional". 

Los niveles de felicidad provienen de una base de datos de 2022 creada por la World Happiness Report. El GDP proviene de datos registrados durante el 2020.
Los datos para la tasa de fertilidad y esperanza de vida fuero obtenidos de Data World Bank, datos correspondinetes a 2022. Los datos para la densidad poblacional se obtuvieron de United Nations Economic Commissions for Europe, datos del 2021.

En la base de datos de Felicidad y GDP se puede encontrar lo siguiente:
- "Pais" (caracteres)
- "GBP" (valor numérico, correspondiente al GDP de cada país).

En la base de datos de Fertilidad se puede encontrar lo siguiente:
- "Pais" (caracteres)
- "FertilityRate" (valor numérico con la cantidad de hijos que podría tener una mujer fertil).

En la base de datos de Esperanza de Vida se puede encontrar lo siguiente:
- "Pais" (caracteres)
- "LifeExpectancy" (valor numérico de la cantidad de años que se espera que vivan los individuos).

En la base de datos de Densidad Poblacional se puede encontrar lo siguiente:
- "Pais" (caracteres)
- "PopulationDensity" (valor numérico de cantidad de personas por kilómetro cuadrado).
